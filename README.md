# ArchiPlus

ArchiPlus is a modern e-commerce web platform where artists and designers can showcase, sell, and manage their creative projects. It connects the creator  with customers through a clean marketplace experience, along with powerful admin tools for managing sales and platform activity.

---

## 🚀 Features

### 👤 Authentication System
- Secure user registration and login
- Role-based access (Admin, Customer)

### 🛒 Marketplace (Customer Interface)
- Browse and view design projects
- Purchase digital design assets
- Save or favorite designs
- Search and filter projects


### 📊 Admin Dashboard
- Overview of total sales and revenue
- Manage users and listed projects
- Monitor platform activity and analytics

---

## 🛠️ Tech Stack

- Frontend: React, HTML, CSS, JavaScript  
- Backend: Node.js / Express  
- Database: SQL  
- Authentication: JWT 
- Other Tools: REST API, Git, Vite
